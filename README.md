# Node Learning

A collection of backend projects and coding exercises built while learning Node.js and modern backend development.

This repository contains multiple mini-projects that demonstrate core backend concepts such as API development, caching, authentication, environment management, and application architecture.

## Goals

The purpose of this repository is to:

- Learn Node.js fundamentals
- Build RESTful APIs with Express.js
- Understand middleware and error handling
- Work with Redis for caching
- Learn Docker and containerization
- Integrate third-party APIs
- Practice backend project structure
- Explore authentication and security concepts
- Improve debugging and problem-solving skills

## Technologies Used

- Node.js
- Express.js
- Redis
- Docker
- JavaScript (ES Modules)
- REST APIs
- dotenv
- Express Rate Limit

## Installation

Clone the repository:

```bash
git clone https://github.com/Ankitsaini406/node-learning.git
```

Move into any project directory:

```bash
cd weather-api
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
